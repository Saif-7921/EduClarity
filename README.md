 📘 Educlarity.AI

> 🚀 AI-Powered Personalized Learning Ecosystem for Indian Students

Educlarity.AI is an intelligent education platform designed to provide **personalized, structured, and adaptive learning experiences** using Generative AI.

It acts as a **24/7 AI mentor**, helping students learn smarter, track progress, and improve performance with real-time insights.

🛠 Installation (Local Setup)

```bash
-> Clone repository
git clone https://github.com/your-username/educlarity-ai.git

➡ Navigate into folder
cd educlarity-ai

➡Install dependencies
npm install

➡Start development server
npm run dev
```

---

🔑 Environment Variables

Create a `.env.local` file:

```env
VITE_GOOGLE_CLIENT_ID=your_google_oauth_client_id
VITE_GEMINI_API_KEY=your_gemini_api_key
```

⚠ Make sure:

* Google Identity Services API is enabled
* Generative Language API is enabled
* OAuth consent screen is configured
